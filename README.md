## 🚀 Project Overview

**InboxPulse Agent** goes beyond basic keyword filtering. Powered by the advanced **Google Gemini API**, this application acts as a smart contextual layer for your emails—analyzing tone, extracting core action items, and dynamically categorizing priorities so you can focus on what actually matters.

### ✨ Key Features
*   **Contextual Analysis:** Deep-semantic understanding of email content using Gemini's language models.
*   **Dynamic Prioritization:** Automatic tagging based on urgency and actionable intent.
*   **Production-Ready Security:** Strict environment isolation preventing API key exposure.
*   **Sleek UI:** Fully interactive, responsive dashboard built with Streamlit.

## 🛠️ Tech Stack

*   **Core Engine:** `Google Gemini API` (via `google-genai`)
*   **Frontend Framework:** `Streamlit`
*   **Data Validation:** `Pydantic`
*   **Environment Management:** `python-dotenv`
